Grocery Shopper Robot

Camera & Lidar Vision Automated Grocery Shopper

Using Webots simulator, python controller file prompts tiago robot to autonomously navigate through the grocery store space using lidar and camera vision, maps and generates efficient path using RRT algorithm, and picks up command items using its color blob detection.

Tearn – iOS Tutoring App

SwiftUI-based application for online tutoring services

Tearn is an iOS application that connects students and tutors in real-time with built-in scheduling, chat, and secure login. The app leverages SwiftUI, OAuth authentication, and Firebase backend.

Stock Sensei

Full Stack Stock Trading Simulator
